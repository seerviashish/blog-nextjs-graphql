# Databases

## POSTGIS

Credential given in .env file

### Start Postgis Server
```
    cd postgis
    sudo docker-compose up -d
```

### Stop Postgis Server
```
    cd postgis
    sudo docker-compose down -d
```

## Redis

http://localhost:8081

For master and slave credential given in docker-compose.yml file.

### Start Redis Server
```
    cd redis
    sudo docker-compose up -d
```

### Stop Redis Server
```
    cd redis
    sudo docker-compose down -d
```
